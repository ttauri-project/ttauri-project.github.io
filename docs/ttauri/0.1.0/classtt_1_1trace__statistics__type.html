Detailed Description

Statistics gathered at the destructor of a trace. This is a wait-free structure.

  • The trace will acquire by incrementing the count, and release when writing the same value in the version.
  • The reading thread acquires by reading the count, then release after it has read the version and checked if they hold the same value.
  • Since there can be traces in multiple threads, it needs to update the statistics themselves atomically as well.
